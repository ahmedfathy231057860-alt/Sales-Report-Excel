Project Overview

The Sales Overall Report Dashboard is an interactive data visualization project built to analyze and present key business performance indicators (KPIs) related to sales, profit, and customer segmentation.
This report consolidates sales performance data across different time periods, categories, and geographic regions, providing actionable insights for data-driven decision-making.
